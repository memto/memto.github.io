### Monitor

```
# check log
$ mysql -u root -p -se "SHOW VARIABLES" | grep -e log_error -e general_log -e slow_query_log
$ mysql -u root -p
$ mysql> SHOW VARIABLES LIKE "general_log%";

$ check connection
$ mysqladmin -u root -p -i 1 processlist
```
